******************************Readme*****************************

Environment setup:
1.	Install Python3 in your system.
2.	Download collections, nltk, pandas, sys, os, io, sys, math Python modules.


Running the code:
1.	Extract the code from the zip file.
2.	Download the corpus and place it in the same folder as the source code.
3.	Open the terminal and go to the source file location by specifying the folder path in the cd command.
4.	Run the source code by running the following command in the terminal:
	python tagger.py corpus_file_path "Sentence to be tested".
5.	The first command line argument represents the file path of corpus that can be a relative or absolute directory path in which the training corpus is present.
6.	The second command line argument represents sentence for which the POS tags to be predicted.
